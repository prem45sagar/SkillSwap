Connection Issues?

If you're having trouble connecting across different networks:

  • 🌐 Make sure both users are using the ngrok URL (not localhost)
  • 🔥 Check if firewall is blocking WebRTC connections
  • 📶 Try switching to a different network or mobile hotspot
  • 🌍 Some corporate networks block peer-to-peer connections
  • ⚡ Consider using Chrome or Edge for better WebRTC support
